2. Módulo de Pasarelas de Pago

El Motor Financiero para el selector del Super Admin.

Estructura de Entornos

Para construir esto correctamente, necesitamos abrir cuentas corporativas en cada plataforma y extraer dos juegos de llaves por cuenta: Uno para el entorno Sandbox/Pruebas y otro para Producción (Dinero Real).

Stripe

Pasarela internacional principal

  • »
    Public Key Publishable key para el frontend.
  • »
    Secret Key Para cobros seguros desde el servidor.
  • »
    Webhook Signing Secret Vital para escuchar cuando un cliente pague en OXXO.

OpenPay

Pasarela nacional (México)

  • »
    Merchant ID
  • »
    Public Key
  • »
    Private Key
  • »
    Webhook Secret Para notificaciones de Paynet y SPEI.

PayPal

Braintree / REST API

  • »
    Client ID
  • »
    Secret Key

Nuvei

Procesamiento alternativo

  • »
    Merchant ID / Site ID
  • »
    Secret Key / Hash Key
